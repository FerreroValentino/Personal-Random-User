RANDOM USER FERRERO VALENTINO

first = primo user,
-> = avanti,
<- = indietro,
last = ultimo user,
tutti si disattivano quando non possono essere selezionati,

scelta genere: 
all = tutti (preselezionata), 
male = solo maschi, 
female = solo femmine,

scelta n user:
input type = barra da 5 a 40 user (preselezionato 10),
ogni volta che la si modifca ricarica gli user,

scelta nazione:
visualizza gli user solo delle nat selezionate,
scleta tra: AU, BR, FR, DE, ES, US,

scelta dimensione:
permette di cambiare la visualizzazione dell'immagine user,
scleta tra: thumbnail, medium (preselezionata), large

cambio tema:
presente una scelta tra tema scuro e chiaro in alto a destra,
luna: scuro,
sole: chiaro.
